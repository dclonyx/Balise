<p> </p>

Définition :

Permet de definir un paragraphe.
Les navigateurs ajoutent automatiquement de l'espace (marge) avant et après chaque élément <p>.
Les marges peuvent être modifiées avec CSS (avec les propriétés de marge).

Exemple :

<p>Ceci est tout le texte qui apparait dans le paragraphe.</p>
