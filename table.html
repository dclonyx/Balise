Définition :

Un tableau HTML est défini avec la balise <table>. Chaque ligne de table est définie avec la balise <tr>. Un en-tête de table est défini avec la balise <th>. Par défaut, les en-têtes de tableau sont en gras et centrés. Une table data / cell est définie avec la balise <td>.</p>

Exemple :
